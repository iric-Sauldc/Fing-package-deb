# Fing-package-deb

Version: 5.4.0 (amd64)

This repository provides the .deb package for Fing Network Scanner version 5.4.0 for 64-bit Debian-based Linux distributions.
This version was previously difficult to find, so it is being shared here for anyone who may still need it. Feel free to use and enjoy it.

Features

Full Fing network scanning functionality

Compatible with most Debian-based systems

Easy installation using dpkg

Installation

Download the package and install it using:

sudo dpkg -i fing_5.4.0_amd64.deb
sudo apt-get install -f

Notes

This is an older Fing release that some users may prefer or require.

Use at your own discretion.
